# Day 1

## What I Built
A JavaScript to-do app using DOM manipulation and event handling. Users can add tasks by clicking a button or pressing the Enter key.

## Features Implemented
- Add tasks dynamically to the page
- Prevent empty tasks from being added
- Clear input after submission
- Add tasks using the Enter key
- Basic debugging using console.log
- Version control using Git and GitHub

## Challenges
Understanding DOM selection, event listeners, and fixing a runtime error caused by incorrect property casing.

## What I Learned
- How JavaScript interacts with HTML through the DOM
- How to handle user input and keyboard events
- How to read and fix JavaScript error messages
- How to commit and push code changes using Git
